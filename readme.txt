== Installation ==

1. Upload the F.B. Feed Plus plugin (`share-me-socials-by-cheeky-apps.zip`) to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Thats it! View any "page" or "post" and you'll see the new floating buttons!
